架構特點

🏗️ 分層設計

系統採用清晰的分層架構，表現層、服務層、離線層、數據層各司其職，便於維護、擴展和測試。

⚡ 實時與批量結合

實時 API 滿足即時決策需求，批量管道支持定期模型訓練和預測，兩者協同運作。

🔄 模型復用

人力審計 API 複用排班預測 API 的模型，避免重複訓練，提高效率。

📊 數據驅動

所有決策基於歷史數據和 AI 模型，確保預測準確性和決策科學性。
