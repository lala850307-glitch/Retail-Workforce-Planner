# Project: Store Labor Allocation Simulation (門市人力配置模擬)

## Overview
This project focuses on simulating operational data and store labor requirements to optimize staffing levels.

## Key Components
- **Data Generation:** Scripts for simulating store and employee data (`data_generator.py`) and operational data (`2025_op_data_generator.py`, `2026Q1_op_data_generator.py`).
- **Simulation Logic:** Core logic for labor simulation (`store_labor_simulation.py`).
- **Standards:** Establishing "Golden Standards" for staffing (`step1_golden_standard.py`).

## Tech Stack
- Python (likely using pandas/numpy for data simulation)

## Development Standards
- Maintain clear variable naming for simulation parameters (e.g., peak hours, staffing ratios).
- Ensure simulation results are reproducible by using fixed random seeds where appropriate.
- Documentation should be in Traditional Chinese where it aligns with existing file naming conventions.
